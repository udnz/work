关于数字的一道趣题

  • 这不是脑筋急转弯，是严肃的数学题！
  • 关键词：数论 质数 素数
  • Author：uonun
  • 2009-3-24

问题：

老师有两个非常聪明的学生，某一天他将这两个学生找来做了一个数学游戏。
老师将自己想好的两个不相等的正整数写在一张纸上藏起来，不告诉任何人。然后分别发给学生甲和学生乙一张白纸，并在学生甲的纸上写下这两个数的积，在学生乙的纸上写下这两个数的和。甲乙两学生只能看自己纸上的数，并不知晓对方纸上写的是几，然后他们进行了如下的对话：

甲：我不知道这两个数分别是多少。
乙：我也不知道这两个数分别是多少。
甲：现在我知道这两个数分别是多少了！
乙：现在我也知道这两个数分别是多少了！

于是，甲和乙分别在自己的纸上写下了这两个数，然后交给老师。老师看了他们交上来的“答卷”，点头说，不错，就是这两个数！

请问，这两个数分别是什么呢？(实际上这两个数可能不唯一，但重点需要的是如何得出这两个数的分析过程。)

分析过程：

令：这两个数为(X,Y)
令：甲得到的积为M，乙得到的和为N。
令：M 可以拆分的表达式集合 {X1*Y1,X2*Y2,X3*Y3...} 为 J。
令：N 可以拆分的表达式集合 {X1+Y1,X2+Y2,X3+Y3...} 为 K。
令：质数集合 {1,3,5,7,11,13,17...} 为 S。

  • 在他们对话之前，我们应当得知：
    1. M 不可能为 1。（最小两个不相等的正整数，他们的积不可能为 1 。）
    2. N 不可能为 2。（最小两个不相等的正整数，他们的和不可能为 2 。）
  • 由第一句对话（甲：我不知道这两个数分别是多少。）得知：
    1. M不可能是 2。（如果 M = 2，则这必定有 X=1,Y=2，甲没有理由不知道这两个数分别是什么。）
    2. 同理，M不可能是 3、4、5。
    3. 更进一步地：M 可以拆分为 X1*Y1,X2*Y2,X3*Y3...，且不唯一（若唯一，则乙必定知道X,Y分别是什么）。（令：这样的表达式集合 {X1*Y1,X2*Y2,X3*Y3...} 为 J）
    4. 同理：M不可能是质数，只可能是合数。即M !∈ S；（如果M是质数，则必定有 X=1,Y=M，甲没有理由不知道这两个数分别是什么）
  • 由第二句对话（乙：我也不知道这两个数分别是多少。）得知：
    1. N ≠ 3。（若 N 为 3，则必定有 X=1,Y=2，乙没有理由不知道这两个数分别是什么。）
    2. 更进一步地：N 可以拆分为 X1+Y1,X2+Y2,X3+Y3...，且不唯一（若唯一，则乙必定知道X,Y分别是什么）。（令：这样的表达式集合 {X1+Y1,X2+Y2,X3+Y3...} 为 K）
  • 由第三句对话（甲：现在我知道这两个数分别是多少了！）得知：
    1. 在前两句对话的结果基础上，甲现在知道这两个数分别是什么了。也就是说，对于 "M 的拆分集合 J"、 "N 的拆分集合 K"，此时他们的交集 J ∩ K 必定有且只有一个元素。
  • 由第四句对话（乙：现在我也知道这两个数分别是多少了！）得知：
    1. 在前两句对话的结果基础上，甲能知道这两个数是什么，也就意味着，甲得出了 "交集 J ∩ K 必定有且只有一个元素" 这个结论。
    2. 此时，乙可以确定地将自己所知的 N，拆分为可知的集合 K，并一一计算集合 K 中的“（X,Y）对”他们各自的乘积，这些乘积中的其中一个必定是 M，再由乘积得到按乘法拆分的另一些“（X,Y）对”，并形成集合 J。看哪一个 J 与 K 满足 "交集 J ∩ K 必定有且只有一个元素"。
      比如：假设乙所知的和为 7。 则有：K = { 3+4，2+5，1+6 }； 如果是 3+4，则甲得到的积应该是12，那么 J = {1*12，2*6，3*4} 如果是 2+5，则甲得到的积应该是10，那么 J = {1*10，2*5} 如果是 1+6，则甲得到的积应该是6，那么 J = {1*6，2*3}
  1. M不可能是质数，只可能是合数。即M !∈ S；
  2. X、Y不可能是质数，只可能是合数。即X !∈ S，Y !∈ S；

1. 假定老师告诉甲的积是2。（最小两个不等的正整数，积是2）

1、2

因此这两个数必定是1和2.
因此我说：我知道这两个数是1和2。		   与对话第一句不符。

甲的思考 乙的思考 我们的思考

积是6有哪些情况？
1、6，2、3
情况不唯一。
因此我说：我不知道这两个数分别是多少。
且告诉乙的和有两种情况：7或5。

  尚符合对话。
    告诉乙的和有两种情况：7或5。
  告诉我的和是：7。
和是7有几种情况？
1、6，2、5，3、4
情况不唯一。
且这三种情况下的积分别有：6、10、12，这三种情况下甲的确无法确定这两个数是什么。
因此我说：我不知道这两个数分别是多少。		 尚符合对话。
  告诉我的和是：5。
和是5有几种情况？
1、4，2、3
情况不唯一。
且这两种情况下积分别有：4、6。
若告诉甲的积是4，则他必定应该知道这两个数是1、4.（不可能为相等的两个数2、2）
可见告诉甲的积一定是6。
因此我说：我知道这两个数分别是多少。		 不符合对话，排除这种情况。
实际上，当甲说他不知道这两个数分别是多少时，就相当于告诉了乙它们的积不可能为4（如果是4，没理由甲不知道它们是1和4）。

他不知道这两个数是什么。
这意味着什么呢？分析下：
我已知：积是6，可能是1、6，2、3。

    
试想若为2、3：
则告诉他的和应当是5。
则他应当猜测这两个数是1、4，2、3。
意即他会猜测告诉我的积是4或6。
我前面说我不知道这两个数是什么，通过这个条件他可以排除积是4这种情况（如果是4，没理由我不知道它们是1和4，我也就没有理由第一次说我不知道了），他就必定能得出告诉我的积是6。他也就必定能得出这两个数是2、3了。与他的回答不符。（他说他不知道）
通过上面的分析，则不可能是2、3这个组合！		   甲自己分析出不可能是2、3这个组合。
这里是甲在以乙的身份思考。
啊！那必定是1、6这个组合！
因此我说：现在我知道这两个数分别是多少了！ 		  甲通过排除，只剩下1、6这个组合了。
仍然符合对话。
  他说他现在知道了。
这意味着什么呢？分析下：
我已知和是7，可能是1、6，2、5，3、4		 上面乙自己的分析已经排除了和是5的情况。
  试想若为：1、6：
则告诉他的积必定是6。
则他应当猜这两个数是2、3，1、6。
意即他会猜测告诉我的和是5或7。
一方面：他猜测告诉我的和是5。
那么加上他已知积是6，他能得出2、3这对数。
他进一步想：当真是2、3的话，他会得出告诉我的和是5，他能肯定：我会得到它们可能是1、4，2、3，即他能肯定我会猜得告诉他的积可能是4和6。
而他自己第一回合说他不知道，他自己也就知道他已经帮我排除了4这种情况。
如此说来，他能肯定我会得到积是6，即他能肯定我能得到这两个数是2、3。但第一回合我告诉他我不知道这两个数是什么。那么他就肯定这两个数不是2、3了。（注1）
另一方面：他猜测告诉我的和是7。
那么加上他已知积是6，他能得出1、6这对数。
他进一步想：（注2）
由此看来，他可以确定这两个数是1、6。
他说他知道了，也正好吻合这个想法！
即，1、6是正确的。
到目前为止，我可以姑且认为我可以确定这两个数了，且看下面的继续分析。

（乙正在分析）

注1：“他进一步想：当真是2、3的话”这一段的分析实质与甲自己的“试想若为2、3”相同，只是身份叠加了一层，让乙站在甲的角度来思考，即：乙在以甲的身份思考甲以自己的身份思考时会发生什么事。

注2：实质思考方式均与注1处的换位思考相同，只不过递归的次数有所增加，且数字对可能性增多，分析的内容几何级加长，因此略去不写。
  试想若为：2、5：
则告诉他的积必定是10。
则他应当猜这两个数是1、10，2、5。
意即他会猜测告诉我的和是11或7。
一方面：他猜测告诉我的和是11。
那么加上他已知积是10，他能得出1、10这对数。
他进一步想：（注2）
另一方面：他猜测告诉我的和是7。
那么加上他已知积是10，他也能得出2、5这对数。
他进一步想：（注2）
由此看来，他是无法确定这两个数的。
他因此应该说他仍然不知道这两个数是什么。
但是事实，他说他知道了，那么此段设想就不正确。
即，排除2、5这个情况。
 （乙正在分析）
  试想若为：3、4：
则告诉他的积必定是12。
则他应当猜这两个数是1、12，2、6，3、4。
意即他会猜测告诉我的和是13、8或7。
一方面：他猜测告诉我的和是13。
那么加上他已知积是12，他能得出1、13这对数。
他进一步想：（注2）
另一方面：他猜测告诉我的和是8。
那么加上他已知积是12，他也能得出2、6这对数。
他进一步想：（注2）
再有：他猜测告诉我的和是7。
那么加上他已知积是12，他也能得出3、4这对数。
他进一步想：（注2）
由此看来，他是无法确定这两个数的。
他因此应该说他仍然不知道这两个数是什么。
但是事实，他说他知道了，那么此段设想就不正确。
即，排除3、4这个情况。		 （乙正在分析）
  三种情况分析完毕，看来只有1、6这种情况符合我们的对话，也就是说，哈，我也知道这两个数是什么了！ 完全符合对话内容！（即不多不少地，正好两个回合得到答案）
    到此，他们都已经知道这两个数了，对话也正好结束了。
即告诉甲的积是6、告诉乙的和是7这样的情况下将完全符合题意和对话内容。
分析完毕。
